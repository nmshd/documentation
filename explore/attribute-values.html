    Relationship Attributes

    The Attribute values in this chapter can only be used in a Relationship Attriubte. Most of them are generic. You can recognize those by the prefix Proprietary (e.g. ProprietaryInteger, ProprietaryString, …). In order to add some validation, you have the option to add valueHints.

    Consent

    Properties

    Name Type Required Validation
    @type "Consent" ✓  
    consent string ✓ max. length: 2000
    valueHintsOverride ValueHintsOverride ✗  
    link string ✗ min. length: 3
    max. length: 1024
    must be a valid URL

    ProprietaryBoolean

    Properties

    Name Type Required Validation
    @type "ProprietaryBoolean" ✓  
    title string ✓  
    description string ✗  
    valueHintsOverride ValueHintsOverride ✗  
    value boolean ✓
